## Exercise 3: Prompt Chain Challenge

### Complex Task: Research and Write

**Break this into a prompt chain:**
"I need a technical blog post about Python async programming for intermediate developers, with code examples, explaining best practices, and including common pitfalls to avoid."

**Suggested Chain:**
1. Research: "Find key concepts about Python async programming..."
2. Outline: "Create an outline for a blog post covering..."
3. Write: "Write the introduction section..."
4. Expand: "Expand section 2 with code examples..."
5. Review: "Review for clarity and technical accuracy..."

## Exercise 5: Common Mistake Fix

### Fix These Prompts:

**1. Too Vague:**
"Help me with my code"

**2. Too Much Information:**
"I need help with my Python project. It's a web scraper. I'm using BeautifulSoup. I'm running Python 3.9. I'm on Windows. I have 16GB RAM. My code is doing something wrong but I'm not sure what. Can you help?"

**3. Multiple Tasks:**
"Write code, test it, document it, create a README, make it production-ready, and write a blog post about it"

**4. No Format:**
"Give me information about neural networks"

**5. Wrong Tone:**
"Explain quantum computing using advanced mathematical notation and assume PhD-level understanding" (for beginners)

## Quick Fire Rounds (5 mins each)

### Round 1: Speed Prompting
Given a task, write the best prompt in 2 minutes. Compare!

### Round 2: Prompt Critique
Show a prompt, identify what's wrong, suggest fixes.

### Round 3: Prompt Battle
Two people get the same task, write different prompts, vote on best!

### Round 4: Prompt Evolution
Start with bad prompt, each person improves it slightly, see the evolution.
